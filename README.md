# Yanjun Chen

Hi, I'm Yanjun, a PhD candidate at The Hong Kong Polytechnic University (joint training with EIT, Ningbo), working on reinforcement learning for LLMs and embodied AI. I make the training environment (reward models, verifiers, curricula) measurable and trainable, judged by what each piece does to the model it trains.

More at **[battam1111.github.io](https://battam1111.github.io)**.

## What I build

- **[C3](https://github.com/Battam1111/C3)**: exact per-decision credit for cooperative LLM agents by transcript replay, with a method-agnostic audit of credit quality. [[paper](https://arxiv.org/abs/2603.06859)]
- **[AccuracyParadox-RLHF](https://github.com/Battam1111/AccuracyParadox-RLHF)**: reproducible RLHF training and evaluation pipelines and reference reward models. [[paper, EMNLP 2024](https://arxiv.org/abs/2410.06554)]
